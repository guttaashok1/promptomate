## Ranking tip

For directory submissions, always include:
- One-line tagline (use the one from the landing page)
- The npm install command (low friction — people can try immediately)
- Link to the demo GIF (once recorded)
- "MIT, open source" prominently — filters people who want free/FOSS tools
