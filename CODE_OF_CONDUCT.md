# Code of Conduct

Promptomate adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

## Short version

- Be kind, inclusive, and professional.
- Focus on what is best for the community.
- Assume good intent; give and accept constructive feedback.
- Behavior that would make someone feel unwelcome is not acceptable.

## Scope

This applies to all project spaces: issues, PRs, discussions, discord (if/when launched), and any external channel where someone is representing the project.

## Reporting

If you see behavior that violates this code, email the repository owner (GitHub profile has the address) or use [GitHub's private reporting flow](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam).

Reports are reviewed within 5 business days and handled confidentially.

## Enforcement

Maintainers may take any action they deem appropriate — from a private warning to a permanent ban — for behavior that violates this code. Decisions will be explained to the person involved.
